## 🛠️ Hướng dẫn cài đặt và chạy
1. Cài đặt thư viện: `npm install`
2. Cấu hình file `.env`.
3. Chạy tất cả kịch bản: `npx playwright test`
4. Xem báo cáo: `npx playwright show-report`
